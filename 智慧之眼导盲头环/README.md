## 接手指南（给 AI 看）

1. **第一步**: 阅读 [00_项目规划/项目规划书.md](./00_项目规划/项目规划书.md) 的第 1~6 章，理解项目全貌
2. **第二步**: 查看规划书第 9 章「待确认问题」，确认是否有与你任务相关的开放项
3. **第三步**: 进入对应工作目录，阅读该目录的 `XX_索引.md` 了解已有内容
4. **第四步**: 完成工作后，更新对应文档并在规划书中标注状态

## 项目基本信息

- **主控**: ESP32-S3
- **雷达**: Ai-Thinker RD03D (24GHz 毫米波)
- **摄像头**: OV2640 (DVP 接口)
- **核心功能**: 毫米波避障 + 盲道/红绿灯/斑马线视觉识别
- **文档版本**: v0.1
