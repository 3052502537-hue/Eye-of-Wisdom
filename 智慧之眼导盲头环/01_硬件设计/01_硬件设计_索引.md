## 当前已确定的信息

> 详见 [项目规划书.md](../00_项目规划/项目规划书.md) 第 3~4 章

### 主控板（ESP32-S3-N16R8）

| 项目 | 参数 |
|------|------|
| CPU | Xtensa® 32-bit LX7 @ 240MHz |
| RAM | 512KB SRAM |
| Flash | 16MB |
| PSRAM | 8MB |
| 外设 | 3×UART, 4×SPI, 34×GPIO, 18×ADC |

### 摄像头板（ESP32-S3-WROOM）

| 项目 | 参数 |
|------|------|
| CPU | Xtensa® 32-bit LX7 @ 240MHz |
| RAM | 512KB SRAM |
| Flash | 8MB |
| PSRAM | 8MB（可选） |
| 外设 | DVP接口, SPI主机, WiFi/BLE |

### 传感器清单

| 传感器 | 数量 | 接口 | 用途 |
|--------|------|------|------|
| SDM10 激光测距 | 1 | UART | 前方精确距离测量（0.05~10m, ±5cm精度） |
| Rd-03D 毫米波雷达 | 2 | UART | 前+后方运动目标检测（0.5~8m, ±60°） |
| OV2640 摄像头 | 1 | DVP | 视觉图像采集（VGA/5-8fps） |

### 反馈设备

| 设备 | 数量 | 接口 | 用途 |
|------|------|------|------|
| 蜂鸣器 | 1 | GPIO47(PWM) | 极危险预警 + 自检提示 |
| WS2812 RGB LED | 1 | GPIO48 | 状态指示（红/黄/绿/蓝） |
| 振动马达 | — | (已舍去) | 触觉反馈方案已移除 |

### 通信接口

| 接口 | 用途 | 参数 |
|------|------|------|
| SPI | 板间通信（主控Master←→摄像头Slave） | 8MHz, Mode0, 2B LE帧 |
| WiFi AP | 主控←→手机APP | 2.4GHz, TCP:8888/UDP:8889 |
| UART1 | SDM10激光 (GPIO17,18) | 460800bps |
| UART2 | 前Rd-03D雷达 (GPIO19,20) | 256000bps |
| UART0 | 后Rd-03D雷达 (GPIO4,5) | 256000bps |
