# 03_代码实现

本目录存放项目源代码。

## 目录结构

```
03_代码实现/
├── src/
│   ├── main.cpp                    # 主程序入口
│   ├── drivers/                    # 硬件驱动层
│   │   ├── rd03d.cpp/h             # RD03D 雷达驱动
│   │   ├── ov2640.cpp/h            # OV2640 摄像头驱动
│   │   ├── buzzer.cpp/h            # 蜂鸣器驱动
│   │   └── vibrator.cpp/h          # 振动马达驱动
│   ├── perception/                 # 感知算法层
│   │   ├── radar_processing.cpp/h  # 雷达数据处理
│   │   ├── lane_detection.cpp/h    # 盲道检测
│   │   └── traffic_light.cpp/h     # 红绿灯识别
│   └── services/                   # 服务层
│       ├── warning_service.cpp/h   # 预警决策
│       └── state_manager.cpp/h     # 状态管理
├── lib/                            # 第三方库
└── test/                           # 测试代码
```

## 开发环境

- **框架**: Arduino-ESP32 或 ESP-IDF（待确认）
- **IDE**: VS Code + PlatformIO（推荐）
- **依赖库**: esp32-camera（摄像头驱动）

## 编码规范

> 详见 [项目规划书.md](../00_项目规划/项目规划书.md) 第 7.3 节

- 变量: 小驼峰或下划线（项目内统一）
- 函数: 动宾结构（`readRadarData()`）
- 宏: 全大写（`WARN_DISTANCE_DANGER`）
- 引脚宏: `PIN_xxx` 格式
