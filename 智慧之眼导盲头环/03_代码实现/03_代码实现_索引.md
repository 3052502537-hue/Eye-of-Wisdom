## 开发环境

### 硬件固件
- **框架**: Arduino-ESP32 + PlatformIO
- **IDE**: VS Code + PlatformIO IDE 插件
- **依赖库**: esp32-camera（摄像头驱动）、FastLED（RGB LED）

### 手机APP
- **框架**: Android Studio + Java
- **依赖库**: TensorFlow Lite（AI推理）、Android TTS（语音播报）

## 编码规范

> 详见 [项目规划书.md](../00_项目规划/项目规划书.md) 第 7.3 节

- 文件命名: 英文小写 + 下划线（如 `sdm10_driver.cpp`）
- 变量: 小驼峰或下划线（项目内统一）
- 函数: 动宾结构（`readLaserData()`）
- 宏: 全大写（`WARN_DISTANCE_DANGER`）
- 引脚宏: `PIN_xxx` 格式

## 快速上手

1. 安装 VS Code + PlatformIO IDE
2. 打开 `firmware_main/` 和 `firmware_cam/` 项目
3. 分别编译烧录到两块ESP32-S3开发板
4. 使用USB-TTL连接串口查看调试信息（115200波特率）
